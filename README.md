## Installation
1. Download `dumBash.sh`:
```
$ git clone https://github.com/MahdyMirzade/dumBash.git ~/dumBash
$ cd ~/dumBash
$ bash setup.sh
```
2. Restart terminal or shell.

## Configuration
Run `config.sh` script:
```
$ cd ~/dumBash
$ bash config.sh
```
